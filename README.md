# Advanced Customizations in Deep Learning and Neural Networks

This repository is organized as a practical, Colab-first guide to advanced deep learning topics in TensorFlow/Keras and PyTorch.

## Project Layout

- Part 1: `Part1_Regularization_Augmentation/`
- Part 2: `Part2_Advanced_Constructs/`

Each module has its own folder-level README with:
- the objective,
- notebook details,
- prerequisites,
- and a dedicated "Video Explanation" section.

Important: videos explaining each Colab are contained in the particular folder of that Colab.

## Part 1: Regularization and Augmentation

See: `Part1_Regularization_Augmentation/README.md`

Topics covered:
- TensorFlow and PyTorch regularization basics
- Monte Carlo dropout and uncertainty
- Weight initialization strategies
- Custom dropout and regularizers
- Keras callbacks, TensorBoard, and Keras Tuner
- KerasCV and multimodal augmentation

## Part 2: Advanced Constructs

See: `Part2_Advanced_Constructs/README.md`

Topics covered:
- Custom learning rate schedules
- Custom losses, activations, initializers, regularizers
- Custom metrics, layers, and model subclassing
- Custom optimizers and custom training loops
- Weights and Biases integration

## How to Use

1. Open any notebook in Google Colab.
2. Run cells in order.
3. Use GPU runtime when needed.
4. Check that folder's README for the related explanation video.

## Notes on Videos

Every notebook folder includes a `README.md` with a "Video Explanation" section.
Add or update that section with your Colab walkthrough video link for the specific notebook.
